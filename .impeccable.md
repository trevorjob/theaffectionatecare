# .impeccable.md — The Affectionate Care Company

## Design Context

### Users
Primarily families and loved ones seeking care services for people with autism, learning disabilities, age-related needs, or alcohol dependency. Secondary audiences: care facilities looking for staffing, and potential staff joining the team. They arrive emotionally invested — searching for someone they can trust with a vulnerable person. The interface must project calm, credibility, and warmth immediately. They are not technically sophisticated; every interaction should feel effortless.

### Brand Personality
**Three words: Calm. Credible. Human.**

Voice is warm but never informal — the tone of a trusted professional who also genuinely cares. Not clinical in the cold sense; clinical in the *private clinic* sense — reassuring, composed, unhurried. The brand speaks with quiet confidence. It never shouts.

Emotional goals: visitors should feel safe, reassured, and seen. Families should feel they're in capable hands. Staff should feel proud to be associated with the brand.

### Aesthetic Direction
**Futuristic pharmacy** — Apple meets a high-end private care clinic. Airy. Light. Simply sophisticated.

- **Base**: White (`#FFFFFF`) and warm off-white (`#FAFAF8`) — never stark, always breathable
- **Primary accent**: Sage / eucalyptus green — the colour of calm, nature, healing
- **Secondary accent**: Champagne / warm sand gold — warmth, quality, restraint
- **Light mode only** — no dark mode
- **Typography**: Playfair Display (headings, display) + Nunito Sans (body, UI text)
- **Motion**: Everything breathes into view. Scroll-triggered fade-ins, gentle parallax, natural cubic-bezier easing. Never abrupt, never spring-bouncy. Motion should feel like inhaling slowly.
- **Anti-references**: No NHS blue, no budget care agency look, no stock-photo overload, no generic rounded card grids, no dark mode, no aggressive CTAs, nothing loud or bouncy.

### Reference Sites / Inspirations
- Apple.com — layout discipline, whitespace, motion restraint
- High-end UK private clinics (e.g. Bupa Cromwell, London Bridge Hospital) — trust signals, typography, calm professionalism
- The blueprint: `site_blueprint_3pages.html` in project root — full section-by-section spec for all 3 pages

```
Fade-in-up:     translateY(24px) → Y(0), opacity 0→1, 0.6s ease
Parallax:       transform only, never layout, willChange: transform
Stagger delay:  100ms between sibling elements

Respect:        prefers-reduced-motion — all animations off
```

### Design Principles

1. **Breathe first.** Whitespace is not empty — it is calm. Every section needs room. Never crowd.
2. **Trust through restraint.** The brand earns confidence by doing less, not more. Every element must justify its presence.
3. **Two accents, used sparingly.** Sage and gold are precious. Overuse kills them. One per section. Never competing.
4. **Motion as exhale.** Animations should feel like the page is gently revealing itself. If it calls attention to itself, it's too much.
5. **Accessibility is dignity.** WCAG 2.1 AA minimum. This site serves vulnerable people and their families — every person must be able to use it.

### Accessibility
- Target: **WCAG 2.1 AA**
- Contrast ratios: minimum 4.5:1 for body text, 3:1 for large text / UI components
- Full keyboard navigation
- Semantic HTML throughout (`main`, `section`, `nav`, `article`, `aside`, etc.)
- All images have descriptive `alt` text
- Focus indicators: visible, styled with sage-400
- Reduced motion: `prefers-reduced-motion: reduce` disables all transitions and animations

### Stack
- Next.js 14 (App Router)
- TypeScript (strict mode)
- Tailwind CSS v3
- Fonts: Google Fonts (Playfair Display + Nunito Sans via `next/font`)
- Animations: CSS transitions + Intersection Observer (no GSAP dependency by default)
